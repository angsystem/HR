ANG HR｜Android assets/HR 同步發布 GitHub Pages

正式前端唯一來源：
app/src/main/assets/HR/

Android 離線備援入口：
file:///android_asset/HR/index.html

GitHub Pages：
1. 將本專案推送到 angsystem/HR 的 main。
2. GitHub → Settings → Pages → Source 選 GitHub Actions。
3. Actions 執行 Deploy ANG HR Pages。
4. 完成後網址維持：https://angsystem.github.io/HR/

注意：OAuth callback 仍使用公開 HTTPS，不可使用 file:///android_asset/。
